# Portfolio Dark UI Template

Template portofolio ini terinspirasi dari desain referensi dengan nuansa dark modern.

## Struktur file
- `index.html` : struktur konten utama
- `styles.css` : styling dan responsive layout
- `script.js` : filter portfolio, mobile menu, dan scroll reveal
- `assets/` : logo, ilustrasi, dan thumbnail project

## Cara edit cepat
1. Ganti nama, deskripsi, dan section resume langsung di `index.html`
2. Ubah daftar project pada variabel `window.PORTFOLIO_PROJECTS` di bagian bawah `index.html`
3. Ganti thumbnail SVG di folder `assets/projects/`
4. Ubah warna tema melalui variabel CSS di bagian `:root` pada `styles.css`

## Menjalankan
Buka `index.html` langsung di browser.
